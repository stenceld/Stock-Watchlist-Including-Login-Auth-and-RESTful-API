Stock Watchlist:
A simple web application for tracking your stock portfolio with real-time price updates from Yahoo Finance.

Live Site
URL: https://stencel.csp-server-class.org

Features
- Add stocks to your watchlist with ticker symbol, quantity, and purchase price
- Automatically fetches current stock prices from Yahoo Finance API
- Displays portfolio with:
  - Individual stock details
  - Current value for each position
  - Profit/loss calculations
  - Total portfolio summary
- Data resides in MySQL database

Technology Stack
- Frontend: HTML, JavaScript
- Backend: PHP
- Database: MySQL
- API: Yahoo Finance
- Hosting: Hostinger

How to Use
1. Visit the site at https://stencel.csp-server-class.org
2. Enter a stock ticker symbol (e.g., AAPL, MSFT, GOOGL)
3. Enter the number of shares you own
4. Enter your purchase price per share
5. Click "Add to Watchlist"
6. View your portfolio with real-time prices and profit/loss calculations

Files
- index.html - Main page with form and watchlist display
- stockHandler.php - Handles form submissions, API calls, and database operations
